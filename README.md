В данной работе с использованием библиотеки NetworkX был построен граф из 50 вершин и выполнен расчёт меры центральности в собственных векторах (Eigenvector Centrality). Структура графа была подобрана таким образом, чтобы значения центральности образовывали характерную форму с локальным минимумом («ямой») и последующим максимумом («горбом»). Для этого центральная группа вершин была организована в виде плотного кластера, обеспечивающего высокие значения центральности, тогда как периферийные вершины и слабо связанные участки графа формировали области пониженной центральности. Расчёт и визуализация результатов позволили проанализировать влияние структуры графа на распределение значений меры центральности и подтвердить наличие требуемой формы распределения.
